# M5Stick C Plus2

## Introduction

Le M5StickCPlus2 est un microcontrôleur ESP32 avec un écran couleur de 1.14 pouces. 

M5StickC PLUS2
SoC : ESP32-PICO-V3-02
Flash : 8MB
RAM : 520KB
PSRAM : 2MB
Battery : 200mAh
Display : 1.14"
Display_TYPE : LCD
LED : RED LEDx1
MIC : SPM1423
SPK : Buzzer
Button : Button x 2
SENSOR : MPU6886
IR : Transmitter
USB-TTL : CH9102
